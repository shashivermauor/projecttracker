🔥 Connect Firebase — Step by Step

Free forever. ~10 minutes to set up. Enables real-time sync + user login.

1
Create Firebase project
Go to console.firebase.google.com → Add project → Name it "worktrack" → Create
2
Enable Firestore Database
Left sidebar → Firestore Database → Create database → Start in test mode → Enable
3
Enable Email/Password Authentication
Left sidebar → Authentication → Get started → Sign-in method → Email/Password → Enable → Save
4
Register a web app & copy config
Project Overview → </> Web icon → Register → copy the firebaseConfig object
5
Paste config in index.html (line ~375)
Find the block marked PASTE YOUR FIREBASE CONFIG HERE and replace the placeholder values. Then re-upload to GitHub.
